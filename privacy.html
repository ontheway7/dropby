개인정보처리방침

시행일: 2026년 3월 16일

1. 개인정보 수집 여부

가는 길에(이하 "앱")는 사용자의 개인정보를 수집하지 않습니다.

2. 위치 정보 사용

앱은 현재 위치를 출발지로 자동 설정하기 위해 기기의 위치 정보를 사용할 수 있습니다. 위치 정보는 앱 내에서만 사용되며 외부 서버로 전송되거나 저장되지 않습니다.

3. 로컬 저장소 사용

앱은 사용자가 저장한 장소 및 경로 정보를 기기 내 로컬 저장소(localStorage)에만 저장합니다. 이 데이터는 외부로 전송되지 않으며 사용자가 직접 삭제할 수 있습니다.

4. 제3자 서비스

앱은 지도 및 경로 탐색을 위해 카카오맵 API 및 카카오모빌리티 API를 사용합니다. 해당 서비스의 개인정보처리방침은 카카오 공식 사이트에서 확인하실 수 있습니다.

5. 문의

개인정보처리방침에 관한 문의는 GitHub 저장소를 통해 연락해 주세요.
